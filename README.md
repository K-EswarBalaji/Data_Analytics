# Data Analytics Projects Portfolio

This repository contains four end-to-end Data Analytics projects that demonstrate data cleaning, exploratory data analysis (EDA), business intelligence, and dashboard development using Python, Excel, and Power BI. Each project focuses on solving real-world business problems by transforming raw data into meaningful insights and actionable recommendations.

# Projects Included

## Project 1: Retail Sales & Profit Analysis

### Objective
Analyze retail sales data to identify sales trends, profitable products, customer segments, regional performance, and business opportunities.

### Tools & Technologies

- Excel
- Power BI

- Data Cleaning & Preprocessing
- KPI Analysis
- Sales Performance Analysis
- Customer Segment Analysis
- Interactive Power BI Dashboard

### Dashboard Insights
- Total Sales
- Total Profit
- Total Orders
- Average Discount
- Shipping Cost
- Sales by Region

- Top 10 Products
